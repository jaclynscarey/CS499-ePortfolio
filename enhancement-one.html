Enhancement One

Software Design and Engineering

For this enhancement, I expanded the complexity of the Recipe Finder App by integrating secure user authentication using Google OAuth. This improvement aligns with the Software Design and Engineering category by demonstrating the ability to scale an application securely while maintaining usability.

Users can now log in using their Google accounts, which enables conditional rendering of user interface elements such as login/logout buttons and access to review functionality. The implementation required frontend state management, secure token handling, and clear navigation feedback, including a login confirmation page.

These changes not only improve the security and maintainability of the app but also showcase real-world software engineering practices used in modern web applications.
